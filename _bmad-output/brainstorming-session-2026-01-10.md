## Ideas Generated

### Core Concept: Escape from Piggy

**High Concept:** Yusuf Baran (IZTECH Rector & Molecular Biologist) navigates through 5 parallel universe versions of campus, escaping wild boars while unlocking DNA-based supernatural abilities.

**Genre:** 2D Pixel Platformer (Celeste-difficulty precision platforming)

**Core Pillars:**
- Tight, responsive Celeste-style controls
- DNA manipulation system (unique hook)
- Parallel universe exploration
- Episode-based narrative with IZTECH/Izmir NPCs
- Mix of standard platforming + running game segments

---

### GAMEPLAY SYSTEMS

#### **Enemy Design: Boar Types**

**Threat Variants:**
- **Chaser Boars** - Relentless pursuit (instant kill in running segments)
- **Patrol Boars** - Predictable patterns for timing challenges
- **Blind Boars** - React to sound/movement
- Additional variants to explore: Tank, Jumping, Exploding types

**Damage System:**
- Platform segments: Boars deal damage (health-based)
- Running segments: One-hit death, instant respawn

### GAMEPLAY STRUCTURE

#### **Episode Flow:**
1. Story dialogue (setup challenge)
2. Platforming section (new mechanics/obstacles)
3. Optional: Running segment
4. Checkpoint with NPC interaction
5. Harder platforming section
6. Episode clear ‚Üí Story beat

#### **Running Segments:**
- Auto-scroll chase sequences
- Boar(s) pursuing from behind/sides
- Obstacles + precise jumps + ability use under pressure
- Instant death on fail, respawn at segment start
- Distributed throughout universes (climactic moments)

#### **Difficulty Design (Celeste-Style):**
- Instant respawns
- Checkpoint every screen
- Death count visible (optional display)
- Challenging through: precision jumps, tight timing, enemy placement
- Tight and responsive movement
- Snappy jump feel
